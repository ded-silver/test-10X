# Test Task 10X - Каталог карточек

Тестовое задание для 10X: карточный каталог с поиском и фильтрацией.

## Технологии

- HTML5
- SCSS (компилируется в CSS)
- Vanilla JavaScript (ES6+)
- Без библиотек и фреймворков

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Компиляция SCSS
npm run sass:build

# Или в режиме отслеживания
npm run sass
```
